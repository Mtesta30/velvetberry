# Velvet Berry · Multi página

Este repositorio contiene el sitio de **Velvet Berry** organizado en páginas independientes para cada sección: inicio, producto, misión y contacto, todo con estilos compartidos y animaciones ligeras.

## ¿Qué hay aquí?

- `index.html`: página de inicio con el héroe, paso a paso y enlaces a las demás páginas.
- `producto.html`: muestra el tarro Velvet Berry, sus beneficios y CTA directo a contacto.
- `mision.html`: detalla la misión, visión y valores del emprendimiento.
- `contacto.html`: pasos para pedir el tarro, WhatsApp, Instagram y el número telefónico.
- `styles.css`: estilos compartidos (paleta, tipografías, animaciones).
- `scripts.js`: Intersection Observer que activa las animaciones a medida que se desplaza la página.

## Cómo desplegar en Vercel

1. Asegura que el contenido esté en `https://github.com/Mtesta30/velvetberry.git`.
2. Importa el repositorio en Vercel usando:
   - **Build Command**: *(déjalo vacío)* porque se trata de archivos HTML estáticos.
   - **Output Directory**: `.`
3. Cada push a `main` disparará un despliegue automático con las cuatro páginas.

## Desarrollo local

Puedes abrir cualquiera de los HTML en el navegador o ejecutar un servidor estático (`npx http-server`, `python -m http.server`, etc.) y navegar entre las páginas mediante los enlaces.
